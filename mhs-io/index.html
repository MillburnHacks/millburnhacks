Tutorials

First, learn how to use Git. It makes it easy to work on code as a team, and provides some other very useful features to boot.

Then, learn the very basics of HTML and CSS. The last language you have to learn is JavaScript. If you already have some programming experience, you can learn from the quick guides LearnXinYMinutes and Babel's guides. If you don't have experience with programming, use Codecademy's tutorial to learn the basics, and then read Babel's guide for the more advanced features.
