Tutorials

We will use Neural Networks and Deep Learning to learn how to use... neural networks and deep learning. It's a very long, mathematical book and will take a few months to go through, but it provides a lot of information. Proper artificial intelligence requires mathematics, so don't try to find another text that hides the math. We will go through each chapter of the book at the meetings, but you should also read it by yourself prior to the meetings so that you can ask any questions you have.
